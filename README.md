# 🌤️ Nimbus — Weather App

A beautifully designed weather app with real-time data and weather-reactive animations.

## 🔗 Live Demo
[willowy-mousse-125f3f.netlify.app](https://willowy-mousse-125f3f.netlify.app)

## ✨ Features
- Real-time weather data for any city
- Dynamic background that changes with weather conditions 
- Rain, snow & storm particle animations
- Glassmorphism UI design
- Humidity, Feels Like & Wind speed stats

## 🛠️ Tech Stack  
- HTML, CSS, JavaScript
- OpenWeather API
- Netlify (Deployment)
